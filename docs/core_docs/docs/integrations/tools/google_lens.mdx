# Google Lens Tool

The Google Lens Tool allows your agent to utilize the Google Lens API to fetch information from images, such as identifying subjects, related images, and links.

## Setup

You will need to sign up for a [SerpApi account](https://serpapi.com/users/sign_up) and get an API key. Then set your API key as `process.env.SERPAPI_API_KEY` or pass it as an `apiKey` constructor argument.
